# VibeCodex Web Template 🌐

Welcome to your new **Next.js** web application, generated by VibeCodex!

This template provides a robust, production-ready starting point with all the modern tools you need to build a scalable full-stack application.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **Auth & Database**: [Supabase](https://supabase.com/)
- **Deployment**: Ready for Vercel

## 📁 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/components`: Reusable UI components.
- `/lib`: Utility functions (e.g., Supabase client).

## 🛠️ Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Environment Variables
Your `.env.local` is already configured with your Supabase credentials! If you need to add more secrets, add them to `.env.local`.

### 3. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
