# VibeCodex Mobile Template 📱

Welcome to your new **Expo React Native** mobile application, generated by VibeCodex!

This template provides a cross-platform (iOS, Android, Web) starting point with all the necessary navigation and authentication setups.

## 🚀 Tech Stack

- **Framework**: [Expo](https://expo.dev/) & React Native
- **Navigation**: [Expo Router](https://docs.expo.dev/router/introduction/) (File-based routing)
- **Auth & Database**: [Supabase](https://supabase.com/)
- **Styling**: Native StyleSheet & safe area contexts

## 📁 Project Structure

- `/app`: The Expo Router directory. Contains `(tabs)` and `(auth)` groups.
- `/components`: Reusable native UI components.
- `/lib`: Utility functions and global singletons (e.g., Supabase client).

## 🛠️ Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Environment Variables
Your `.env` is already configured with your `EXPO_PUBLIC_SUPABASE_URL` and `ANON_KEY`.

### 3. Start the Bundler
```bash
npx expo start
```
Use the Expo Go app on your physical device, or press `i` to open an iOS simulator, and `a` for an Android emulator.
