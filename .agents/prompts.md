# VibeCodex Agent Prompts 🤖

This file contains powerful AI prompts that you can copy & paste into Anthropic Claude, OpenAI ChatGPT, Gemini, or GitHub Copilot to accelerate your VibeCodex project development.

## 1. Database & Edge Functions (Supabase)

**Prompt: Generate a schema with RLS**
> "I am using a VibeCodex template with Next.js and Supabase. Please generate a complete database schema in SQL for a [describe your app/entity e.g., 'task management'] system. Include primary keys, foreign keys linking to `auth.users`, and Row Level Security (RLS) policies that only allow authenticated users to view, create, update, and delete their own records."

**Prompt: Create a Supabase Edge Function**
> "Write a Supabase TypeScript Edge Function that takes a payload of [describe inputs], processes it by [describe logic], and updates the Supabase database using the `@supabase/supabase-js` client. Include instructions on how to use `supabase secrets set` and `supabase functions deploy`."

## 2. Front-End Components (Next.js / Tailwind)

**Prompt: Create a Dashboard View**
> "I am using Next.js App Router and TailwindCSS inside a VibeCodex project. Create a modern, beautiful 'Dashboard' server component that fetches user data from Supabase. Use a dark mode aesthetic with purple and blue gradients (`from-purple-500 to-blue-500`). Then create a client component inside it that renders a list of cards for this data."

**Prompt: Dynamic Interactive Form**
> "Using React Hook Form and Zod for validation, build a contact form for my Next.js VibeCodex app. The form should have fields for Name, Email, and Message. When submitted, display a loading state and simulate an API call. Style it beautifully using TailwindCSS with glassmorphism effects (`bg-white/10 backdrop-blur-md border border-white/20`)."

## 3. Mobile Screens (Expo Router)

**Prompt: Add a new Tab Screen**
> "I am building an Expo application using Expo Router generated by VibeCodex. I need to add a new tab to my `app/(tabs)` directory called `Settings`. Provide the code for `settings.tsx` including a header title 'Settings', user profile info, and a 'Sign Out' button that calls `supabase.auth.signOut()`."

**Prompt: React Native Animations**
> "Using `react-native-reanimated`, write a generic React Native component that wraps its children and performs a smooth 'fade-in and slide-up' animation when the screen mounts. This will be used in my VibeCodex Expo app."

## Pro Tips for AI Coding
- **Context is Key**: Always mention you are using Next.js App Router, TailwindCSS, Expo Router, and Supabase.
- **Copy-Paste Errors**: If you encounter a TypeScript or lint error, simply paste the exact error message into the AI.
- **Component Splitting**: Ask the AI to split large components into smaller, reusable client and server pieces.
